콜라쥬 핀치 줌 테스트 랩

메인 페이지와 분리된 독립 테스트 창입니다. HTML 조각에서 이미지 URL을 파싱하거나 직접 사진을 올린 뒤, 같은 캔버스 미리보기에서 모바일 핀치 줌과 드래그를 확인할 수 있습니다.

1. HTML 파서

`img[src]`, `source[srcset]`, `meta[property="og:image"]`를 읽어서 이미지 후보를 만듭니다.

2. 직접 업로드

원격 이미지가 막히는 경우를 대비해 업로드만으로도 바로 미리보기와 핀치 제스처를 시험할 수 있습니다.

캔버스 바깥이 아니라 각 이미지 슬롯 안에서 제스처를 시작해야 반응합니다.
