관리자 페이지(준비)

권한별 로그인은 서버/인증 시스템을 통해야 안전하게 운영할 수 있습니다.

지금 바로 구현할 수 있는 권장 방식

  • 마스터(본인): 이메일(예: Gmail) 기반 OAuth 로그인
  • 운영진(관리자): 마스터가 사용자 이메일을 승인해 역할 부여
  • 게시글 작성: 관리자 인증 후에만 글쓰기 버튼 노출/쓰기 API 허용

이 방식이면 공용 비밀번호를 여러 사람이 공유할 필요가 없어지고, 내가 비밀번호를 계속 외울 필요가 없습니다.

현재 단계(임시)

지금은 공개 페이지 + 게시판 뼈대만 구성되어 있습니다.

다음 단계에서 인증/권한 + 게시글 저장 API를 붙이면 바로 실서비스로 전환됩니다.
